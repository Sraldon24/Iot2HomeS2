DomiSafe is a real-time home monitoring and security system designed to track environmental conditions, movement, and device status. It integrates hardware sensors, Raspberry Pi processing, a cloud database, and a unified dashboard for control and monitoring.

The system is built using a modular Python backend, Flask web dashboard, MQTT-based remote control, and a hardware security system running on Raspberry Pi. Motion events, environment logs, and camera captures are saved locally and mirrored to the cloud.
